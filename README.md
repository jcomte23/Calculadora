Calculadora web clasica con un diseño moderno para hacer las 4 operaciones basicas de la matematica

## tecnologias

- html
- css
- javascript

## levantamiento en local

1. clonar el proyecto

```
git clone git@github.com:jcomte23/Calculadora.git
```


2. abre la carptea

```
code nombreProyecto
```

3. correlo en local, por ejemplo con go live y luego abre el navegador

```
http://localhost:3000
```
